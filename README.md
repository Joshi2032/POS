# Zapata Flutter (scaffold)

This is a minimal Flutter scaffold port of the Angular `Zapata` project to bootstrap conversion work.

Quick start

1. Install Flutter SDK: https://flutter.dev/docs/get-started/install
2. From this folder run:

```bash
flutter pub get
flutter run -d <device> # e.g. android, windows
```

Notes
- This is an initial automated scaffold with placeholder pages and simple widgets.
- Continue converting UI and business logic from `d:/Zapata/src` into `lib/` pages/widgets.
